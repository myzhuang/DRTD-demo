
# RGB-Event Modality Fusion with Annotation Code for DRTD

This code is used to visualize the RGB-Event fusion results while displaying the annotations.  
It demonstrates that the calibrated RGB and event modalities can be aligned and share a unified annotation file.

Environment:
Python 3.9
Required libraries: os, glob, cv2

Steps:
1. Run the fusion_with_annotation_process.py script to generate fusion results with annotation.
The input data is rgb and event images and annotation files. 
The output data is the fusion results with annotation. 

Commands:
  ```bash
  python fusion_with_annotation_process.py
  ```
